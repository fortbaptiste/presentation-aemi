Contexte

Qu'est-ce que AEMI ?

AEMI est une application de gestion de maintenance industrielle (GMAO) développée pour le Forrest Groupe en République Démocratique du Congo.

En termes simples, c'est un outil qui aide les équipes à suivre, planifier et gérer toutes les interventions de maintenance sur les équipements industriels — principalement des groupes électrogènes.

L'application permet de savoir quel équipement a besoin de maintenance, qui s'en occupe, et où en est chaque intervention.

Correction du Scroll Mobile

BUG CORRIGÉ

Le problème

Sur mobile, quand on ouvrait un tableau détaillé dans la section Reporting, on restait bloqué à l'intérieur du tableau sans pouvoir remonter dans la page. C'était très frustrant pour les utilisateurs.

La solution

Nous avons remplacé les conteneurs de scroll imbriqués (un scroll dans un scroll) par un seul conteneur de scroll. Maintenant, le défilement est fluide partout, même dans les tableaux détaillés.

📱

AVANT

Scroll bloqué dans les tableaux, impossible de remonter

📱

APRÈS

Correction des Accents

Le français, c'est important !

Les graphiques et labels affichaient des mots sans accents. Par exemple, "Preventive" au lieu de "Préventive", "Completee" au lieu de "Complétée". Nous avons créé une fonction automatique qui corrige ces erreurs partout dans l'application.

⚙ Solution technique : Une fonction normalizeAccents() est appliquée automatiquement à tous les textes et labels de l'application.

Scroll Mobile Corrigé

Une expérience fluide sur téléphone

Le scroll sur mobile est maintenant fluide partout. Plus de blocage dans les tableaux. Le problème venait de conteneurs de défilement imbriqués : un élément scrollable à l'intérieur d'un autre élément scrollable.

La solution : nous avons remplacé ces imbrications par un seul conteneur de scroll principal. Les tableaux s'adaptent maintenant à la largeur de l'écran ou permettent un scroll horizontal sans bloquer le scroll vertical de la page.
